testsuite::test_DoubleVector_IntVector.sli - test vector operations
Synopsis: 
(test_DoubleVector_IntVector) run ->
Description: 
 
   
  This test checks the basic operations on IntVector and DoubleVector objects. Tested are  
  - conversion and creation  
  - arange, ones and zeros  
  - put and get  
  - Map and forall  
  - add, mul
