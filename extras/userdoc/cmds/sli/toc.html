Name:
toc - Return wall clock time elapsed since tic was called.  
  Synopsis:  toc -> runtime
FirstVersion: 
2003-07-29
Description: 
 
  Determines the wall clock (real) time elapsed since tic was called  
  last, in seconds.  Resolution is given by pclockspersec.
