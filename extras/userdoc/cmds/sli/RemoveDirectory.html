Command: RemoveDirectory

NEST HelpDesk Command Index NEST Quick Reference
Name:
RemoveDirectory - Delete a directory
FirstVersion: 
Oct 12th 1999
Parameters: 
string : Name of subdirectory to be deleted.  
               boolean : A flag if operation succeded.
Author: 
Hehl
Bugs: 
-
Synopsis: 
string RemoveDirectory -> boolean
Examples: 
(RemoveMeDir) RemoveDirectory -> True, and  
              ./RemoveMeDir does not longer exist, provided it ever did  
              and was empty.
Remarks: 
-
Description: 
RemoveDirectory deletes the named subdirectory of your  
      current working directory, provided it is empty.
