Command: dup

NEST HelpDesk Command Index NEST Quick Reference
Name:
dup - Duplicate the object which is on top of the stack
Synopsis: 
any dup -> any any
Diagnostics: 
Raises StackUnderflow error if the stack is empty.
Author: 
docu edited by Sirko Straube
Examples: 
2 dup -> 2 2  
  (hello) dup -> (hello) (hello)
