Name:
Directory - Return current working directory
FirstVersion: 
Oct 12th 1999
Parameters: 
string : Name of current working directory
Author: 
Hehl
Bugs: 
-
Synopsis: 
Directory -> string
Examples: 
Directory = -> /home/MyAccount/SNiFF/synod2
Remarks:
Description: 
Returns name of current working directory. This is where all ls,  
  filestream etc. operations are done per default.
